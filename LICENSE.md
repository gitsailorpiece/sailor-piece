## 🔧 Common Issues & Fixes

**❌ Issue: GUI doesn't appear after executing the script**
> **Fix:** Wait 30–50 seconds after clicking Execute — the hub needs time to load all modules. If it still doesn't appear, make sure you are fully in-game (not on the loading screen) before executing. Try pressing **Right Shift** or **;** to toggle GUI visibility.

**❌ Issue: Kill Aura isn't attacking enemies**
> **Fix:** Ensure Kill Aura is toggled **ON** in the Combat section of the GUI. Check that the aura radius slider is set above zero. Some enemies require you to be at a minimum level — check the level requirement displayed next to enemy names.

**❌ Issue: Executor says "Script Error" or "Attempt to index nil"**
> **Fix:** Your script version is outdated. Re-download the latest loadstring from this repository. This usually happens after a Sailor Piece game update changes internal variable names.

**❌ Issue: Auto Boss not looping / stops after one kill**
> **Fix:** Make sure the "Loop" toggle is enabled in the Auto Boss submenu. If the boss fails to spawn, try the "Force Summon" option. Ensure your in-game Beli is sufficient for the summon cost.

**❌ Issue: Devil Fruit Sniper teleports but fruit is already gone**
> **Fix:** Enable "Server Hop" mode in the Fruit Sniper settings — this cycles through servers to find one where the fruit is still on the ground. Increase teleport speed in settings if your connection is slow.

**❌ Issue: Script crashes on Android / Arceus X**
> **Fix:** Ensure you are running the latest version of Arceus X or Delta Mobile. Close all background apps before executing to free up RAM. Re-download the mobile loadstring from this repository — do not use outdated pastebin copies.

**❌ Issue: Anti-AFK not working — getting kicked anyway**
> **Fix:** Make sure Anti-AFK is toggled on *before* going idle, not after. Some Roblox servers use server-side AFK detection — in that case, enable "Auto Move" in the Anti-AFK submenu for a supplemental solution.
